PowerTools Pro Website Preview

Note: This is a preview page for the PowerTools Pro website. The actual website consists of 5 HTML pages with responsive design using Tailwind CSS and JavaScript.

To view the actual website pages, please open the individual HTML files directly in your browser:

  • Home Page
  • Services Page
  • Portfolio Page
  • Contact Page
  • About Us Page

Project Structure

├── index.html (Home Page) ├── services.html (Services Page) ├── portfolio.html (Portfolio Page) ├── contact.html (Contact Us Page) ├── about.html (About Us Page) ├── css/ │ └── styles.css (Custom CSS styles) ├── js/ │ └── main.js (JavaScript functionality)

Pages Overview

Home Page (index.html)

The home page features:

  • Navigation menu with responsive mobile design
  • Hero section with call-to-action
  • Features section highlighting key benefits
  • Services preview with icons and descriptions
  • Client testimonials carousel
  • Newsletter subscription form
  • Footer with contact information and quick links

Services Page (services.html)

The services page includes:

  • Detailed service offerings for power tools, hand tools, and accessories
  • Information about battery holding machines and spares
  • AI tool recommendation section
  • Service pricing and packages
  • Call-to-action for consultations

Portfolio Page (portfolio.html)

The portfolio page showcases:

  • Filterable portfolio grid with project categories
  • Project images with hover effects
  • Modal windows for detailed project information
  • Client testimonials related to projects
  • Call-to-action for custom projects

Contact Page (contact.html)

The contact page provides:

  • Contact form with validation
  • Business contact information with icons
  • Google Maps integration
  • Business hours
  • FAQ section with accordion functionality
  • Social media links

About Us Page (about.html)

The about us page contains:

  • Company story and history
  • Mission and vision statements
  • Core values with icons
  • Company milestones timeline
  • Team member profiles
  • Certifications and partner logos
  • Call-to-action section

Technologies Used

  • HTML5: For structure and content
  • Tailwind CSS: For responsive styling and layout
  • JavaScript: For interactive elements like carousels, modals, and form validation
